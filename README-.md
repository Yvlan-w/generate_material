# 营销素材生成小程序

## 项目背景

市场部同事日常制作营销海报，经常面临两个痛点：一是找设计师沟通成本高，一张图来回改好几次，耗时1-2天很常见；二是合规审核环节，投资行业对宣传内容要求严格，人工审核容易遗漏风险点。

基于这个痛点，我们开发了一个小程序工具，让市场同事能够自己快速生成合规的营销图片。

***

## 使用方式

操作非常简单，不需要学习成本：

1. 打开小程序，用聊天的方式描述需求
2. 系统会问几个关键问题（目标客户、风格偏好等），回答即可
3. 等待大约1分钟，图片自动生成
4. 如果不满意，直接说修改意见，系统会重新生成

举个实际使用的例子：

```
我：帮我做一张产品介绍的海报
系统：好的，请问这张海报主要面向什么客户？
我：高净值客户
系统：希望用什么颜色和风格？
我：蓝色，专业一点的风格
系统：明白了，正在为您生成...
```

***

## 主要功能

### 1. 对话式需求收集

不用填表格，不用写需求文档，像聊天一样就能说清楚想要什么。系统会主动问关键信息，确保生成的图片符合要求。

### 2. 支持参考图片

看到喜欢的设计风格，可以上传参考图，告诉系统想借鉴哪些方面（比如颜色、布局、整体感觉），系统会照着做。

### 3. 自动合规检查

投资行业对宣传内容有很多限制，比如不能出现"稳赚不赔"、"年化收益XX%"这类表述。系统会自动检查生成的内容，不合规的会打回重新生成，避免违规风险。

### 4. 图片微调

第一次生成的图片如果不满意，不用重新描述全部需求，直接说哪里需要改就行，比如"颜色太暗了"、"背景太复杂"，系统会根据反馈调整。

### 5. 图片管理

所有生成的图片都会自动保存，方便后续查看和使用，也支持按状态筛选（合规通过/待审核）。

***

## 适用场景

这个工具主要用于以下场景：

- 朋友圈营销素材（产品推广、活动宣传）
- 公众号配图（文章封面、数据图表）
- 内部培训材料（课程PPT配图）
- 线下活动物料（海报、展架设计）

我们对这个工具的预期是：

| 指标       | 现状   | 预期     |
| -------- | ---- | ------ |
| 单张海报制作周期 | 1-2天 | 1分钟左右  |
| 设计师沟通成本  | 较高   | 基本无需沟通 |

简单来说，就是把原来需要几天的工作压缩到几分钟完成，同时降低合规风险。
